### COMANDOS BASÍCOS

Autenticar via computador local no Github:
```
gh auth login
```

Clonar repositoório no Github:
```
gh repo clone nome-do-repositório
```

Abrir VS Code:
```
code .
```

### Salvar no Github

Preparar arquivos para salvar:
```
git add.
```


Comando commit -m "mensagem"
```


Baixar atualizações do Github

git pull
```

Enviar para Github:
```
git push
```
